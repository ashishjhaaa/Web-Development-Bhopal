CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It separates content from design.

  • Inline CSS: Style added directly to an HTML element.
  • Internal CSS: Style added inside a <style> tag in the <head> section.
  • External CSS: Style written in a separate .css file and linked with <link>.

CSS Syntax

      selector {
                property: value;
            }

Ways to Add CSS (Examples)

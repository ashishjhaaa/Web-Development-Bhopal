1. animation-name

Links the element to keyframes.

Default: none

Move

2. animation-duration

Controls how long one animation cycle takes.

Default: 0s (animation won't run)

Slow

3. animation-delay

Wait time before animation starts.

Default: 0s

Wait

4. animation-iteration-count

Number of times animation repeats.

Default: 1

Loop

5. animation-direction

Controls forward / backward movement.

Default: normal

Alternate

6. animation-timing-function

Controls speed curve of animation.

Default: ease

Smooth

7. animation-fill-mode

Controls styles before/after animation.

Default: none

Freeze

8. Animation Shorthand

Used in real projects.

Pro

1. transition-property

What it does: Tells CSS which property should animate.

Common values: all (default), width, height, background-color, transform

Default value: all

Hover

2. transition-duration

What it does: Defines how long the transition takes.

Common values: s, ms (1s, 500ms)

Default value: 0s (no animation)

Hover

3. transition-timing-function

What it does: Controls the speed curve of animation.

Common values:

ease (default) - slow → fast → slow

linear - same speed

ease-in - slow start

ease-out - slow end

ease-in-out - slow start & end

Default value: ease

Hover

4. transition-delay

What it does: Wait time before transition starts.

Common values: s, ms (1s, 300ms)

Default value: 0s

Hover

Transition Shorthand (Industry Usage)

Order: property → duration → timing-function → delay

Example: transition: all 1.2s ease-in-out 0s;
